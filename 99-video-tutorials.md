## How to use these videos

1. **First pass - Microsoft Learn Foundry overview** (one sitting). Maps Foundry projects, model catalog, deployments, prompt flow, agents, evaluation.
2. **Second pass - AI Show + Azure Friday** episodes for any topic where you didn't feel confident (RAG, agents, function calling, content safety, evaluation).
3. **Third pass - RAG + agents walkthroughs** so the patterns are muscle memory. AI-300 leans heavy on "given this requirement, which Foundry component?".
4. Then take the **Practice Assessment** (see the sidebar) before scheduling the real exam.

> Tip: AI-300 questions almost always include 2-3 *technically correct* options. The trick is *fit* - provisioned throughput vs. pay-as-you-go, embeddings model choice, when to use Agent Service vs. just chat completions + tools, hybrid vs. vector search. Watching the Microsoft framing in these videos is exactly the framing the exam uses.
